ТЗ № 2.3 (по JS)

Допустим, параметры http-запроса хранятся как свойства объекта.
Напишите функцию сериализации параметров в query-строку с добавлением к произвольному url.

Файл js

Результат сериализации параметров отличается от jQuery.param

  • Функции не вычисляются
  • Для массивов лишние элементы не создаются

Выполните в консоли следующий код:

var test = { 'o': {}, 'O': new Object, 'a': [], 's': 'str', 'S': new String('qqq'), 'n': 123, 'N': new Number(456), 'b': true, 'B': new Boolean(true), 'd': new Date, 'f': function(){this.poo = 90; return 1}, 'F': new Function('return 1'), 'r': /\d/, 'R': new RegExp('\d+', 'g') }; getUrl('http://example.com:8080/path/to/script.php?a=15#uuu', test);
